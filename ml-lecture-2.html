Classification vs Regression

Second Lecture - 3 Minutes

Click to start the lecture

Two Main Types of Supervised Learning

Key Question: What type of output are we predicting?

  • Classification: Predicts categories (Yes/No, Cat/Dog/Bird)
  • Regression: Predicts numbers (Price, Temperature, Age)

Classification: Predicting Categories

Email Spam Detection Example

Classification Characteristics:

  • Output is a category or class
  • Examples: Spam/Not Spam, Cat/Dog, Pass/Fail
  • Uses decision boundaries to separate classes
  • Accuracy measured by correct classifications

Regression: Predicting Numbers

House Price Prediction Example

Regression Characteristics:

  • Output is a continuous number
  • Examples: Price, Temperature, Stock Value
  • Finds best-fit line/curve through data
  • Error measured by distance from actual values

Key Differences Summary

🎯 Classification

Output: Categories/Classes

Examples:

  • Email: Spam or Not Spam
  • Image: Cat, Dog, or Bird
  • Medical: Disease or Healthy

Algorithms: Decision Trees, SVM, Naive Bayes

📈 Regression

Output: Continuous Numbers

Examples:

  • House Price: $425,000
  • Temperature: 23.5°C
  • Stock Price: $150.75

Algorithms: Linear Regression, Random Forest
