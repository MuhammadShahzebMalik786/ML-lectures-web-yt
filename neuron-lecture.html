üíª Simple Implementation

import numpy as np

class Neuron:
    def __init__(self, n_inputs):
        self.weights = np.random.randn(n_inputs) * 0.1
        self.bias = 0

    def relu(self, x):
        return max(0, x)

    def sigmoid(self, x):
        return 1 / (1 + np.exp(-x))

    def forward(self, inputs, activation='relu'):
        z = np.dot(self.weights, inputs) + self.bias

        if activation == 'relu':
            return self.relu(z)
        elif activation == 'sigmoid':
            return self.sigmoid(z)
        return z

# Usage
neuron = Neuron(3)
inputs = [1.0, -0.5, 2.0]
output = neuron.forward(inputs, 'relu')
print(f"Output: {output}")

üéØ When to Use What?

ReLU: Hidden layers (fast, simple)
Sigmoid: Binary output (0 or 1)
Softmax: Multi-class output (probabilities)

Example Network:

  • Input ‚Üí Hidden (ReLU) ‚Üí Hidden (ReLU) ‚Üí Output (Sigmoid/Softmax)
  • Image classification: ReLU hidden layers + Softmax output
  • Binary prediction: ReLU hidden layers + Sigmoid output

üéØ Key Takeaways

  • Neurons are the building blocks of neural networks
  • Weights determine input importance
  • Activation functions add non-linearity
  • ReLU for hidden layers, Sigmoid/Softmax for output
  • Multiple neurons create powerful networks
