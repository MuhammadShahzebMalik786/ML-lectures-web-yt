ü§î What is Unsupervised Learning?

Machine learning that finds hidden patterns in data without labeled examples. It discovers structure in data where we don't know the "right answer".

Supervised: Input ‚Üí Algorithm ‚Üí Predicted Output (with known labels)

Unsupervised: Input ‚Üí Algorithm ‚Üí Hidden Patterns (no labels)

üéØ K-Means Algorithm Deep Dive

The most popular clustering algorithm that partitions data into k clusters by minimizing within-cluster distances.

K-Means algorithm in action: Initialize ‚Üí Assign ‚Üí Update ‚Üí Repeat

üìã Algorithm Steps:

  1. Initialize: Place k random centroids
  2. Assign: Each point goes to nearest centroid
  3. Update: Move centroids to cluster centers
  4. Repeat: Until centroids stop moving

Distance Formula: d = ‚àö[(x‚ÇÅ-x‚ÇÇ)¬≤ + (y‚ÇÅ-y‚ÇÇ)¬≤]

Centroid Update: New center = Average of all points in cluster

üíª Python Implementation

from sklearn.cluster import KMeans
import numpy as np
import matplotlib.pyplot as plt

# Generate sample data
X = np.random.rand(100, 2) * 10

# Apply K-Means
kmeans = KMeans(n_clusters=3, random_state=42)
labels = kmeans.fit_predict(X)
centroids = kmeans.cluster_centers_

# Plot results
plt.scatter(X[:, 0], X[:, 1], c=labels, cmap='viridis')
plt.scatter(centroids[:, 0], centroids[:, 1],
c='red', marker='x', s=200)
plt.title('K-Means Clustering')
plt.show()

‚öñÔ∏è K-Means Pros & Cons

‚úÖ Advantages

  • Simple and fast
  • Works well with spherical clusters
  • Scales to large datasets
  • Guaranteed to converge
  • Memory efficient

‚ùå Disadvantages

  • Must choose k beforehand
  • Sensitive to initialization
  • Assumes spherical clusters
  • Affected by outliers
  • Struggles with varying sizes

K-Means convergence: Centroids find optimal positions for clustering

üîß Choosing the Right K

The biggest challenge in K-Means is selecting the optimal number of clusters.

Elbow Method: The "elbow" point indicates optimal number of clusters

Methods to Find Optimal K:

  • Elbow Method: Plot cost vs k, find the "elbow" point
  • Silhouette Score: Measures how well points fit their clusters
  • Gap Statistic: Compares clustering with random data

Silhouette Analysis: Higher scores mean better clustering quality

# Elbow Method
costs = []
for k in range(1, 11):
    kmeans = KMeans(n_clusters=k)
    kmeans.fit(X)
    costs.append(kmeans.inertia_)

plt.plot(range(1, 11), costs, 'bo-')
plt.xlabel('Number of Clusters (k)')
plt.ylabel('Cost')
plt.title('Elbow Method')
plt.show()

üöÄ Key Takeaways

  • K-Means is the go-to clustering algorithm
  • Choose k using elbow method or silhouette analysis
  • Works best with spherical, similar-sized clusters
  • Always standardize your data before clustering
  • Try multiple random initializations for best results
